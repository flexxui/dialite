Welcome to Dialite's documentation!
===================================

Dialite is a small library to show simple dialogs to the user,
without the need for a heavy GUI toolkit.

Introduction
------------

.. automodule:: dialite


Dependencies and installation
-----------------------------

Dialite is pure Python and has no dependencies. It works on Python 2.7 and 3.4+.
Install using: ``pip install dialite``.
